# ariofengenden.com

Personal website for Ari Ofengenden, served via GitHub Pages at
[ariofengenden.com](https://ariofengenden.com).

- `index.html` — the site (single page, self-contained styles)
- `assets/headshot.jpg` — portrait
- `CNAME` — custom-domain configuration for GitHub Pages

Editing: change the text in `index.html` and push to `main`; GitHub Pages
redeploys automatically within a minute or two.
